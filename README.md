# ElegantWeatherApp

A clean weather dashboard built with vanilla HTML, CSS, and JavaScript. Gets live weather from the Open-Meteo API — no API key needed.

## Features

- Search any city with autocomplete suggestions
- Current weather with temperature, conditions, humidity, wind
- 6-day forecast
- Weather icons update based on conditions
- Dark blue theme, responsive layout

## How to use

Open index.html in a browser. That's it. No build step, no API keys.

## API

Uses Open-Meteo (free, no key required) and a city search API for location lookups.
